**测试项**: 校规数据完整性  
**现象**: `type object 'SchoolRule' has no attribute 'is_active'`  
**位置**: `database/models/school_rule.py`  
**影响**: 无法正确筛选活跃校规，可能导致已删除校规仍被检索  
**建议**: 
1. 检查模型定义是否缺少`is_active`字段
2. 或修改测试代码使用正确的字段名（可能是`status`或其他）

**现象**: 正确率: 0.0% (0/4)

**影响**: 权限隔离可能存在漏洞

2. 验证创建分组方案时是否正确设置创建者ID

### 问题5: Student模型缺少user_id字段

**测试项**: 敏感数据访问模式  
**现象**: `type object 'Student' has no attribute 'user_id'`

**建议**: 
1. 检查模型定义是否缺少`user_id`字段
2. 或确认学生登录功能是否通过其他方式实现账号关联
